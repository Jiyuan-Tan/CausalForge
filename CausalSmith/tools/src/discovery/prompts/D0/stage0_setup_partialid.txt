=== GLOBAL SETUP — Partial identification anchor ===

Observable / unobservable structure:
- Let P denote the observable distribution and θ the unobserved (target) parameter. State the dimensions and the map (Y, A, X, Z, ...) → P precisely.
- The identified set Θ_I(P) is the set of θ values consistent with both P and the maintained restrictions. State Θ_I(P) explicitly.

Restrictions:
- Each identifying restriction (shape: monotone-treatment-response (MTR), monotone-treatment-selection (MTS), monotone-IV (MIV); IV-style: exclusion, mean-independence; sensitivity: bounded log-odds ratio, bounded marginal sensitivity Λ; missing-data: bounded outcome support, MAR-perturbation) gets its own labeled assumption. Do NOT bury a restriction in a proof step.

Bounding functional:
- Define the lower and upper bounds (θ_L(P), θ_U(P)) explicitly as functionals of P. The point of the derivation is to prove `Θ_I(P) ⊆ [θ_L(P), θ_U(P)]` (outer bound) or `Θ_I(P) = [θ_L(P), θ_U(P)]` (sharp bound) under named restrictions.

Sharpness vs outer bound:
- Distinguish "outer bound" (every θ in Θ_I lies in the interval, may be loose) from "sharp" (every point in the interval is attainable by some θ ∈ Θ_I). A bound that is only outer is still publishable but must be flagged as such.

Proof technique:
- **Use any technique that fits the problem.** No technique is required or preferred — including methods from other fields (information theory, functional analysis, optimal transport, etc.) or brand-new constructions. The catalogue below is a suggestion menu, freely ignorable:
    * extreme-point / LP / convex-program arguments (Manski; Balke–Pearl; MST MTR-LP);
    * shape-restriction algebra (MTR / MTS / MIV) and Manski worst-case completion;
    * copula / Fréchet–Hoeffding bounds and extremal couplings (Tchen; Fan–Park);
    * random-set theory, capacity functionals, Aumann / Choquet integrals (Beresteanu–Molchanov–Molinari);
    * support-function / dual-variational characterizations;
    * optimal-transport extremal couplings (Galichon);
    * completeness / Tikhonov inverse / bridge-function identification (Newey–Powell; Miao–Geng–Tchetgen-Tchetgen);
    * sensitivity duality, breakdown frontiers, ε-contamination algebra (Tan; Masten–Poirier; Rambachan–Roth);
    * moment-inequality projections (Andrews–Soares; Chernozhukov–Hong–Tamer);
    * quantile / CDF envelopes; shape-constrained density characterizations (Blomquist et al.).
- The only hard requirement is that each step is traceable: state which technique each step uses (whether or not it appears on the suggestion menu) and cite a published source when one exists. If two distinct techniques produce the same bound from different primitives, that is a feature, not redundancy — surfacing both clarifies what the restriction is really doing.
