TASK. Find every NAMED operator, predicate, function class, law family, or rule
symbol that a STATEMENT body (theorem / lemma / assumption / definition) USES but a
reader CANNOT resolve. A symbol is resolvable iff some environment DEFINES it (its
title names it, or its body says "we say … when", "the class … is", "denote … by",
"$X := …$"). The defining environment must occur before later uses. Do not flag
a later use merely because it omits a cross-reference.

FLAG, per symbol, the worst applicable case:
- `undefined` — used in a statement but NO environment defines it (it is introduced
  only in passing, or its definition is deferred to section prose that has no
  anchor a `\cref` can target). E.g. a class `\mathcal H^\beta(…)` or an operator
  `\mathrm{overlapDen}` used in assumptions/lemmas with no defining environment.
- `wrong-ref` — the use carries a `\cref{obj:X}` but env X does NOT define the symbol
  (e.g. referencing the boundary-extrapolator definition for the Hölder class).
- `mismatch` — the defining statement and the use disagree on the symbol's
  parameters/arguments (defined as `\mathcal H^\beta(L;[0,t_0])` but used as
  `\mathcal H^\beta(C_\beta;[0,t_0])` with no statement that the constants coincide).
- `no-anchor` — defined only in an environment-less location so no `\cref` can reach
  it (a named class/operator should be its own `definitionv`, or an anchored,
  labelled definition, not a floating prose sentence).

EXCLUDE standard mathematics no reader needs defined: `\mathbb R`, `\mathbb E`,
`\Pr`, `\mathbb 1`, `\vee`/`\wedge`/`\le`/`\to`, Greek parameters (κ, β, λ, …),
ordinary functions (e, μ, g), and the ambient covariate space `\mathcal X` IF the
setup/observable-law definition introduces it. Flag only genuinely NAMED objects a
reader of this paper could not resolve.
