For a given \(\mu\in\mathcal V_{n,l}\), the active outcome-pair set \(E_\mu\), all armwise minimizer sets \(R_{ab,z}(\mu)\), and the active-normal sum \(s_\mu\) are computable in polynomial bit time without listing \(2^l n^2\) types. The normalized base \(\mathcal K_\mu=\{h\in T_\mu:s_\mu^{\mathsf T}h=1\}\) is compact and has the polynomial-size extended formulation already displayed in the proof of \(\text{thm:explicit-global-parent-map}\). Consequently one can compute one extreme ray selected by any rational linear objective with deterministic polynomial-time lexicographic linear programming. For any such ray, separable piecewise-linear line search computes the exact finite neighboring endpoint in polynomial bit time, and evaluating whether \(P(\nu)=\mu\) is polynomial-time. The global parent graph is already acyclic, rooted at \(\lambda^\star\), and reaches every vertex. Thus only complete polynomial-delay generation of all relevant projected extreme rays or children, including a polynomial exhaustion certificate, remains open.
