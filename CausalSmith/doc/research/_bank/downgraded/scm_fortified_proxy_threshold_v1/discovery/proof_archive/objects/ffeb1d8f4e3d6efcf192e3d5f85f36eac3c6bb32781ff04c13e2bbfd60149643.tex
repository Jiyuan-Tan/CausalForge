At \(\gamma=K\), Definition \(\mathrm{def:dimension-functions}\) and the binomial theorem give
\[\begin{aligned}h_{K,m}(K)&=m^K+\sum_{j=1}^{K}\binom Kj(m-1)^j\\&=m^K+\bigl((1+m-1)^K-1\bigr)=2m^K-1.\end{aligned}\]
Because \(m^{K-K}=1\), it follows that
\[D_{K,m}(K)=h_{K,m}(K)+1=2m^K,\qquad Q_{K,m,u}(K)=2u,\]
and hence \(\Delta_{K,m,u}(K)=2(m^K-u)\). If \(u\le m^K\), the endpoint slack is nonnegative, so the defining set in Definition \(\mathrm{def:dimension-frontier}\) contains \(K\) and \(\gamma_{\mathrm{dim}}(K,m,u)<+\infty\). Conversely, if \(u>m^K\), then the endpoint slack is negative. Proposition \(\mathrm{prop:slack-frontier}\) says that slack is strictly increasing in \(\gamma\), so \(\Delta_{K,m,u}(r)<\Delta_{K,m,u}(K)<0\) for every \(r<K\); the defining set is empty and \(\gamma_{\mathrm{dim}}(K,m,u)=+\infty\). This proves the equivalence. Finally, if \(u=m^K\), then \(\Delta_{K,m,u}(K)=0\), while strict monotonicity gives \(\Delta_{K,m,u}(r)<0\) for every \(r<K\). Therefore the first nonnegative-slack cardinality is exactly \(K\).
