\paragraph{9-fragment-statement.}
\paragraph{Motivating claim, restated verbatim.}
Under Assumptions~\ref{ass:finite-memory}--\ref{ass:clean-window}, \ref{ass:regular-row}, \ref{ass:boundary-limits}, \ref{ass:bounded-response}, and \ref{ass:nondegenerate-target}, suppose \(K=K_T\) is deterministic and \(p_T(K)>0\).
There are constants \(0<c_K<C_K<\infty\), depending only on the event-window template and the omitted pre-period convention, such that the nonzero singular values of the event-window FWL Gram block generated by \(E_k,C_{10},C_{11}\) satisfy
\[
c_K\Lambda_T(K)\le \sigma_j(G_T(K))\le C_K\Lambda_T(K)
\]
for every event-window direction whose clean switch-on row is represented, while every direction orthogonal to \(R_T(K)\) has singular value \(o(\Lambda_T(K))\).
Consequently, if \(\Lambda_T(K)/\kappa_T\to\infty\), then \(\ell_K\in\mathcal V_T^{\mathrm{reg}}(K)\) for every causal switch-on ATT loading \(\ell_K\) supported on \(K_{\mathrm{ATT}}\); if \(\Lambda_T(K)/\kappa_T\to\lambda\in(0,\infty)\), then the retained limiting row space is exactly the span of the clean-window directions indexed by \(J^{\mathrm{bdry}}(K,\lambda)=\{j:s_j(K)\lambda\ge1\}\), so partial identification occurs precisely when this set is a nonempty proper subset of the clean-window directions; and if \(\Lambda_T(K)/\kappa_T\to0\), no nonzero causal switch-on event-window loading is regularly identified.

\paragraph{Refutation summary.}
Refuted: for \(K=\{-1,0\}\) and \(a_T=b_T=T^{-1/2}\), the block direction \(E_0+C_{11}\) has two-way residualized Gram scale at least \(T/64\), while \(\Lambda_T(K)\asymp T^{1/2}\).

\paragraph{13-refutation-attempt.}
\paragraph{Primitive 1: finite-\(T\) grid.}
I first use the finite-grid primitive with \(T\in\{2,3,4\}\), binary treatment paths, and the event-window template \(K=\{-1,0\}\) with the single pre-period row omitted.
For every Markov law with \(0<a_T,b_T<1\), every binary path has positive probability on this grid.
The clean event row is the switch-on indicator \(E_{0,it}=\one\{t\ge2,D_{i,t-1}=0,D_{it}=1\}\); the nuisance rows are \(C_{10,it}=\one\{D_{i,t-1}=1,D_{it}=0\}\) and \(C_{11,it}=\one\{D_{i,t-1}=1,D_{it}=1\}\).
This bounded grid does not by itself refute the asymptotic phase-transition statement, because a fixed finite grid can always be covered by constants that depend on the template.
It does, however, expose the candidate obstruction:
\[
E_{0,it}+C_{11,it}
=D_{it}\qquad\text{for every }t\ge2.
\]
Thus the FWL block generated by \(E_0,C_{10},C_{11}\) contains, up to the single boundary period \(t=1\), the treatment-state column \(D_{it}\), which is not a rare switch-on count.

\paragraph{Primitive 2: first-order Markov assignment.}
Now take the stationary symmetric Markov sequence
\[
a_T=b_T=\delta_T=T^{-1/2},\qquad
\rho_T=1-2\delta_T,\qquad
\Pr(D_{it}=1)=\Pr(D_{it}=0)=\frac12,
\]
and keep \(K=\{-1,0\}\).
All locked support conditions used by the conjecture hold: \(0<a_T,b_T<1\) for large \(T\), \(p_T(K)=\phi_T=\delta_T/2>0\), and
\[
\Lambda_T(K)=n_T(K)p_T(K)=(T-1)\frac{\delta_T}{2}\asymp T^{1/2}.
\]
The response surface can be any bounded response vector with a nonzero switch-on target row, because the contradiction is entirely in the assignment-design Gram block.

Let \(d_{it}=D_{it}\), and let \(M_F\) denote residualization on unit and calendar-time fixed effects.
Since the Markov law is stationary and symmetric, the population calendar-time mean of \(d_{it}\) is \(1/2\) at every \(t\).
Therefore the two-way residual of the treatment-state column is \(d_{it}-\bar d_i\), where \(\bar d_i=T^{-1}\sum_{t=1}^T d_{it}\), and
\[
\E\sum_{t=1}^T (M_F d_{it})^2
=
\E\sum_{t=1}^T (d_{it}-\bar d_i)^2
=
\frac{T}{4}-\frac{\operatorname{Var}\!\left(\sum_{t=1}^T d_{it}\right)}{T}.
\]
For the stationary two-state Markov chain,
\[
\operatorname{Cov}(d_{it},d_{i,t+h})=\frac14\rho_T^h.
\]
Hence
\[
\frac{\operatorname{Var}\!\left(\sum_{t=1}^T d_{it}\right)}{T}
=
\frac14+\frac{1}{2T}\sum_{h=1}^{T-1}(T-h)\rho_T^h
\le
\frac14+\frac12\sum_{h=1}^{\infty}\rho_T^h
\le
\frac{1}{4\delta_T}.
\]
With \(\delta_T=T^{-1/2}\), this gives
\[
\E\sum_{t=1}^T (M_F d_{it})^2
\ge
\frac{T}{4}-\frac{\sqrt T}{4}
\ge
\frac{T}{8}
\qquad (T\ge4).
\]

The actual design direction in the conjectured Gram block is \(q_{it}=E_{0,it}+C_{11,it}\).
It equals \(d_{it}\) for every non-boundary period \(t\ge2\), and differs from \(d_{it}\) only at \(t=1\).
Residualization is a contraction, so
\[
\|M_F q\|_2
\ge
\|M_F d\|_2-\|M_F(d-q)\|_2
\ge
\sqrt{T/8}-1.
\]
For all sufficiently large \(T\), \(\|M_F q\|_2^2\ge T/32\).
Let \(u=(1,0,1)^\transpose\), corresponding to the block direction \(E_0+C_{11}\).
For the FWL Gram block \(G_T(K)\),
\[
u^\transpose G_T(K)u=\|M_F q\|_2^2\ge \frac{T}{32},
\qquad
\|u\|_2^2=2,
\]
so the largest nonzero singular value obeys
\[
\sigma_{\max}(G_T(K))\ge \frac{u^\transpose G_T(K)u}{\|u\|_2^2}\ge \frac{T}{64}.
\]
But \(\Lambda_T(K)\asymp T^{1/2}\).
Thus
\[
\frac{\sigma_{\max}(G_T(K))}{\Lambda_T(K)}
\ge c\sqrt T\to\infty
\]
for a numerical constant \(c>0\).
No constant \(C_K\) depending only on the event-window template and the omitted pre-period convention can satisfy \(\sigma_j(G_T(K))\le C_K\Lambda_T(K)\) for all nonzero singular values of the block generated by \(E_k,C_{10},C_{11}\).
This contradicts the first singular-value assertion of Conjecture~\ref{conj:markov-event-rank}; the advertised point/partial/nonregular consequences therefore do not follow from the stated assumptions.
