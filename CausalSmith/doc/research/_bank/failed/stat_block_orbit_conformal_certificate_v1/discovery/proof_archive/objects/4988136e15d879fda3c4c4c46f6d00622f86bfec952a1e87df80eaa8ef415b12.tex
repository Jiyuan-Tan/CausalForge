Fix an arm \(a\) and condition on \(\mathcal T\). As in \(\texttt{thm:orbit-balance-iff}\), there is a common within-orbit score kernel \(F_{a,O}\). The calibration-score and target-score laws are therefore \[G_a=\sum_{O\in\mathcal O}q_{a,O}F_{a,O},\qquad H_a=\sum_{O\in\mathcal O}p_{a,O}F_{a,O}.\] For every Borel set \(C\), \[|H_a(C)-G_a(C)|\le\frac12\sum_{O\in\mathcal O}|p_{a,O}-q_{a,O}|=\Delta_a(w),\] where the displayed total-variation bound follows equivalently by maximally coupling the two finite orbit draws and then using the same conditional draw from \(F_{a,O}\) whenever the orbits agree.

Keep the \(M\) calibration scores fixed in law and replace the target score by an independent draw from \(G_a\). In this reference experiment all \(M+1\) scores are iid, so the lexicographic-rank argument from \(\texttt{thm:common-selector-lp}\) gives reference failure probability at most \(\alpha/2\). Changing only the target marginal from \(G_a\) to \(H_a\) changes the probability of any event involving it and the independent calibration vector by at most \(\Delta_a(w)\). Consequently, \[\Pr(S_{0,a}>r_{a,(k)}\mid\mathcal T)\le\alpha/2+\Delta_a(w).\] If neither arm fails, then \(\tau_T\in\mathcal C_w\). A union bound, which remains valid for the common and possibly dependent target lifts, yields \[\Pr_P(\tau_T\in\mathcal C_w\mid\mathcal T)\ge1-\alpha-\Delta_0(w)-\Delta_1(w).\]

Finally, \(\sum_O\delta_{a,O}(w)=0\) because both orbit vectors are probability vectors. Hence \[\sum_O[\delta_{a,O}(w)]_+=\sum_O[-\delta_{a,O}(w)]_+=\frac12\sum_O|\delta_{a,O}(w)|=\Delta_a(w).\] Each positive or negative summand is bounded by this common total, proving \(|\delta_{a,O}(w)|\le\Delta_a(w)\) for every orbit.
