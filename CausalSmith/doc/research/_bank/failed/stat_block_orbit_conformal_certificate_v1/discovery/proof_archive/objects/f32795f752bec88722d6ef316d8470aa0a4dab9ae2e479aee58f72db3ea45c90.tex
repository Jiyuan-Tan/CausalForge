Put \(N=M+1\). For either arm \(c\), \[k_c\le M\Longleftrightarrow \lceil N(1-\varepsilon_c)\rceil\le N-1\Longleftrightarrow \varepsilon_c\ge 1/N.\tag{1}\] The invariance, boundedness, and experiment-membership argument is exactly the orbit-constant construction proved in \(\texttt{prop:disjoint-orbit-witness}\), and it concerns the actual \(\mathcal G\)-orbits. Every selected calibration score in both arms is zero. Consequently, \[r_{c,(k_c)}=0\quad\text{if }k_c\le M,\qquad r_{c,(k_c)}=+\infty\quad\text{if }k_c=M+1.\tag{2}\] The common target effect is \(K\) when \(a=1\) and \(-K\) when \(a=0\). If both ranks are finite, (2) gives \(I^{\boldsymbol\varepsilon}_{0,w}=I^{\boldsymbol\varepsilon}_{1,w}=\{0\}\), hence \(\mathcal C^{\boldsymbol\varepsilon}_w=\{0\}\) and coverage zero. If exactly one rank is \(M+1\), \(\texttt{prop:asymmetric-rank-boundary}\) and (2) give one interval \([-K,K]\) and one interval \(\{0\}\); their Minkowski difference is \([-K,K]\), which contains either possible target effect. If both ranks equal \(M+1\), the same proposition gives \([-2K,2K]\), again with coverage one. Equation (1) proves the rank equivalence. On the equal-split diagonal the two ranks coincide, leaving precisely the two regimes in \(\texttt{prop:disjoint-orbit-witness}\). As there, a pair-orbit instantiation is legitimate only when the corrected local pair-action condition holds.
