Put \(N=M+1\). Because \(N\) is an integer, the elementary ceiling identity \(\lceil N-x\rceil=N-\lfloor x\rfloor\) gives, for each arm \(a\),
\[
k_a=\lceil N-N\varepsilon_a\rceil=N-\lfloor N\varepsilon_a\rfloor.
\]
Consequently, by \(\texttt{def:finite-rank-error}\),
\[
\rho_{M,\varepsilon_a}=\frac{N-k_a}{N}=\frac{\lfloor N\varepsilon_a\rfloor}{N}\le\varepsilon_a.
\tag{1}
\]
For an integer \(j\ge0\), the equality \(\lfloor N\varepsilon_a\rfloor=j\) holds if and only if
\[
j\le N\varepsilon_a<j+1,
\]
which is equivalent to \(\varepsilon_a\in[j/N,(j+1)/N)\). This proves the coordinatewise staircase and locates every elbow. Moreover, \(k_a=N\) if and only if \(\lfloor N\varepsilon_a\rfloor=0\). Since \(\varepsilon_a>0\), this is equivalent to \(0<\varepsilon_a<1/N\).

Finally specialize to \(\varepsilon_0=\varepsilon_1=\alpha/2\). The definitions give
\[
k_0=k_1=\left\lceil N\left(1-\frac\alpha2\right)\right\rceil=k
\]
and, by \((1)\),
\[
\rho_{M,\varepsilon_0}=\rho_{M,\varepsilon_1}=\frac{\lfloor N\alpha/2\rfloor}{N}=\rho_{M,\alpha}.
\]
Thus the old one-dimensional staircase is precisely the diagonal section of the two-dimensional one.
