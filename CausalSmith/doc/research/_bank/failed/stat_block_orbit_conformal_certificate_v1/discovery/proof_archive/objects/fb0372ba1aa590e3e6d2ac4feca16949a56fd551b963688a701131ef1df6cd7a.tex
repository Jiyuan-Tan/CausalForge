Set \(N=M+1\). For every integer \(N\) and real \(x\), the elementary ceiling--floor identity \(\lceil N-x\rceil=N-\lfloor x\rfloor\) follows by multiplying \(\lfloor x\rfloor\le x<\lfloor x\rfloor+1\) by \(-1\) and adding \(N\). Taking \(x=N\alpha/2\) gives \[N-k=N-\left\lceil N-\frac{N\alpha}{2}\right\rceil=\left\lfloor\frac{N\alpha}{2}\right\rfloor.\] Division by \(N>0\) proves the two formulas for \(\rho_{M,\alpha}\), and \(\lfloor x\rfloor\le x\) proves \(\rho_{M,\alpha}\le\alpha/2\). Moreover, \(\lfloor N\alpha/2\rfloor=j\) if and only if \(j\le N\alpha/2<j+1\), which is equivalent to \(2j/N\le\alpha<2(j+1)/N\). Finally, \(k=N\) if and only if \(N-k=0\), hence if and only if \(\lfloor N\alpha/2\rfloor=0\). Under \(0<\alpha<1\), the latter is exactly \(0<\alpha<2/N\).
