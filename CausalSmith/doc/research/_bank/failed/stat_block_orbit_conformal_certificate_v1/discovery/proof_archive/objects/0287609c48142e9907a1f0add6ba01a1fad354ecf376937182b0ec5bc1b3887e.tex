For any \(w\in\Delta(\mathcal H)\), the definition of \(A\) gives, row by row, \[(Aw)_{a,O}=\sum_{h\in\mathcal H}w_h\sum_{u\in O}q^{\mathrm{obs}}_{h,a}(u)=q^{\mathrm{obs}}_{w,a}(O).\] The corresponding coordinate of \(b_{\mathrm{LP}}\) is \(q_{\mathrm{tar},a}(O)\). Therefore \(Aw=b_{\mathrm{LP}}\) is exactly joint orbit balance. Since \(w\in\Delta(\mathcal H)\) if and only if \(w_h\ge0\) for every \(h\) and \(\sum_h w_h=1\), the stacked linear program is feasible exactly when a jointly balancing common mixture exists. Moreover, \(Aw=\sum_h w_hA_{\cdot h}\), so feasibility is equivalent to \[b_{\mathrm{LP}}\in\operatorname{conv}\{A_{\cdot h}:h\in\mathcal H\}.\]

For the infeasibility certificate, let \(C=\operatorname{conv}\{A_{\cdot h}:h\in\mathcal H\}\). This finite convex hull is compact. If \(b_{\mathrm{LP}}\notin C\), choose \(x_*\in C\) minimizing Euclidean distance to \(b_{\mathrm{LP}}\), and set \(y=b_{\mathrm{LP}}-x_*\ne0\). For every \(x\in C\), minimality along the segment \(x_*+t(x-x_*)\), \(0\le t\le1\), implies \(y^\top(x-x_*)\le0\). Consequently \[\sup_{x\in C}y^\top x\le y^\top x_*<y^\top b_{\mathrm{LP}},\] where the strict gap is \(\lVert y\rVert_2^2\). This is the asserted finite-dimensional Farkas separating hyperplane.

Now fix a feasible \(w\) and again condition on \(\mathcal T\). By \(\texttt{thm:orbit-balance-iff}\), for each arm separately, \(S_{1,a},\ldots,S_{M,a},S_{0,a}\) are iid. Append independent continuous tie breakers as specified in \(\texttt{def:rank-inversion}\). The tie-broken rank of \(S_{0,a}\) among the \(M+1\) scores is uniform on \(\{1,\ldots,M+1\}\). If \(S_{0,a}>r_{a,(k)}\), then this rank is greater than \(k\); hence \[\Pr(S_{0,a}>r_{a,(k)}\mid\mathcal T)\le\frac{M+1-k}{M+1}\le\frac{\alpha}{2}.\] The display also holds when \(k=M+1\), because then the left side is zero. The last inequality follows from \(k=\lceil(M+1)(1-\alpha/2)\rceil\). If neither armwise failure occurs, then \(Y_{0T}(a)\in I_{a,w}\) for both arms and therefore \(\tau_T\in I_{1,w}-I_{0,w}=\mathcal C_w\). Bonferroni's inequality, with no arm-independence requirement, now yields \[\Pr(\tau_T\in\mathcal C_w\mid\mathcal T)\ge1-\alpha.\]
