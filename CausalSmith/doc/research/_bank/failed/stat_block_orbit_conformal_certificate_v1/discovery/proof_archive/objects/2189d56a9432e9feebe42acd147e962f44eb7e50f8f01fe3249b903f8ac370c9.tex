Put \(N=M+1\), write \(w_j=\lambda(O_j)\), and set \(W=\sum_{j=0}^{M}w_j\). Under \(\mathsf P\), the random orbit \(O_0\) has \(p_{O_0}>0\) almost surely. The overlap hypothesis gives \(q_{O_0}>0\), hence \(w_0>0\) and therefore \(W>0\) under \(\mathsf P\otimes\mathsf Q^M\).

For a deterministic configuration \((O_j,R_j)_{j=0}^{M}\), define \(t_j\) to be the left-continuous \((1-\varepsilon)\)-quantile of
\[
\mu_j=\frac{w_j\delta_{+\infty}+\sum_{\ell\ne j}w_\ell\delta_{R_\ell}}{W}.
\tag{1}
\]
On configurations with \(W=0\), set \(t_j=+\infty\). Let \(A=\{j:R_j>t_j\}\). We claim the deterministic inequality
\[
\sum_{j=0}^{M}w_j\mathbf 1\{j\in A\}\le\varepsilon W.
\tag{2}
\]
The claim is immediate if \(A=\varnothing\) or \(W=0\). Otherwise choose \(j_*\in A\) for which \(R_{j_*}\) is minimal among \(\{R_j:j\in A\}\). Because \(R_{j_*}>t_{j_*}\), the definition of the left-continuous quantile implies that the \(\mu_{j_*}\)-mass strictly below \(R_{j_*}\) is at least \(1-\varepsilon\). The atom indexed by \(j_*\) is at \(+\infty\) in \(\mu_{j_*}\), so
\[
\sum_{\ell:R_\ell<R_{j_*}}w_\ell\ge(1-\varepsilon)W.
\tag{3}
\]
Consequently \(\sum_{\ell:R_\ell\ge R_{j_*}}w_\ell\le\varepsilon W\). Minimality of \(R_{j_*}\) on \(A\) gives \(A\subseteq\{\ell:R_\ell\ge R_{j_*}\}\), which proves (2). This also treats ties: the reported event is the strict exceedance of a weakly included quantile.

The common fibre laws \(F_o\) imply
\[
\frac{d\mathsf P}{d\mathsf Q}(o,r)=\lambda(o)
\quad\mathsf Q\text{-almost surely}.
\tag{4}
\]
Let \(\mathbb E_{\mathsf Q^N}\) denote expectation when all \(N\) orbit--score pairs are independent with law \(\mathsf Q\). Changing measure only in coordinate zero yields
\[
\Pr_{\mathsf P\otimes\mathsf Q^M}\{R_0>t_0\}
=\mathbb E_{\mathsf Q^N}\!\left[w_0\mathbf 1\{R_0>t_0\}\right].
\tag{5}
\]
The product law \(\mathsf Q^N\) is exchangeable, and the map that sends a configuration and a distinguished index \(j\) to \(t_j\) is permutation equivariant. Thus every summand below has the same expectation, and (2) gives
\[
\begin{aligned}
\Pr_{\mathsf P\otimes\mathsf Q^M}\{R_0>t_0\}
&=\frac1N\mathbb E_{\mathsf Q^N}\!\left[\sum_{j=0}^{M}w_j\mathbf 1\{R_j>t_j\}\right]\\
&\le\frac{\varepsilon}{N}\mathbb E_{\mathsf Q^N}[W]
=\varepsilon\,\mathbb E_{\mathsf Q}[\lambda(O)]
=\varepsilon.
\end{aligned}
\tag{6}
\]
The last equality follows from \(\sum_o q_o\lambda(o)=\sum_o p_o=1\), with null coordinates omitted. Under \(\mathsf P\otimes\mathsf Q^M\), the threshold \(t_0\) is exactly \(\widehat r\), so (6) proves the lemma. The argument is the finite-support likelihood-ratio weighted-exchangeability proof: infinity augmentation makes the unknown target score unnecessary, overlap makes (4) valid, and the strict event in (2) makes the result valid with ties and zero calibration weights.
