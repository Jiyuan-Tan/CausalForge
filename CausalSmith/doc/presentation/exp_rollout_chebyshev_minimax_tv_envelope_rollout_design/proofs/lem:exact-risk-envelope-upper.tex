\begin{proof}[Proof of \cref{obj:lem:exact-risk-envelope-upper}]
Fix an admissible schedule \(p\in S_{k,q}\), and write \(s=\sigma_0^2/n\). Since \(\sigma_0^2\ge0\), \(s\ge0\). For any admissible schedule \(p'\in S_{k,q}\), \cref{obj:lem:unbiased-weight-set-nonempty} gives at least one \(w\in W_\beta(p')\). % lean: unbiased_weight_set_nonempty
For such a fixed \(p'\), consider the set of exact risks obtained by choosing \(w\in W_\beta(p')\), namely
\[
\left\{
\sup_{P\in\mathcal P_\beta}
\operatorname{Var}_\pi\!\left(\sum_{j=0}^k w_j\bar Y_j\right):
w\in W_\beta(p')
\right\},
\]
and the corresponding set
\[
\left\{
\left(\sum_{j=0}^k |w_j|\right)^2:
w\in W_\beta(p')
\right\}.
\]
The first set is nonempty and bounded below by \(0\), because variances are nonnegative; the second set is nonempty by the preceding paragraph. % lean: fixedRisk_nonneg
For every element of the second set, represented by some \(w\in W_\beta(p')\), \cref{obj:lem:variance-envelope-sharpness} applied to every law in \(\mathcal P_\beta\) gives
\[
\operatorname{Var}_\pi\!\left(\sum_{j=0}^k w_j\bar Y_j\right)
\le
s\left(\sum_{j=0}^k |w_j|\right)^2 .
\]
Taking the supremum over \(P\in\mathcal P_\beta\) preserves this bound. % lean: variance_envelope_sharpness
Therefore the infimum over admissible weights satisfies
\[
\inf_{w\in W_\beta(p')}
\sup_{P\in\mathcal P_\beta}
\operatorname{Var}_\pi\!\left(\sum_{j=0}^k w_j\bar Y_j\right)
\le
s\,
\inf_{w\in W_\beta(p')}
\left(\sum_{j=0}^k |w_j|\right)^2
=
\frac{\sigma_0^2}{n}A_\beta(p').
\]
This is the order-theoretic comparison of the two nonempty infima under the pointwise envelope bound. % lean: sInf_le_mul_sInf_of_forall_exists_le
Applying this fixed-schedule conclusion to the given \(p\) proves
\[
\inf_{w\in W_\beta(p)}
\sup_{P\in\mathcal P_\beta}
w^\prime\Gamma_P(p)w
\le
\frac{\sigma_0^2}{n}A_\beta(p).
\]
% lean: exact_risk_envelope_upper

It remains only to pass from a fixed schedule to the nested risk. The set of admissible schedules is nonempty because it contains the given \(p\). For every \(p'\in S_{k,q}\), the fixed-schedule bound just proved gives
\[
\inf_{w\in W_\beta(p')}
\sup_{P\in\mathcal P_\beta}
w^\prime\Gamma_P(p')w
\le
\frac{\sigma_0^2}{n}A_\beta(p').
\]
Taking the infimum over \(p'\in S_{k,q}\), again using \(s=\sigma_0^2/n\ge0\), yields
\[
R_{\mathrm{exact}}(\beta,k,q)
\le
\frac{\sigma_0^2}{n}
\inf_{p'\in S_{k,q}} A_\beta(p')
=
\frac{\sigma_0^2}{n}M_{\beta,k,q}.
\]
% lean: sInf_le_mul_sInf_of_forall_exists_le
\end{proof}
