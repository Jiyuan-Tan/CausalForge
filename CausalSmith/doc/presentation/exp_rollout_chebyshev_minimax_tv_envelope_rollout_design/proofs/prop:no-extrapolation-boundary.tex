\begin{proof}[Proof of \cref{obj:prop:no-extrapolation-boundary}]
Let
\[
w_0=-1,\qquad w_\beta=1,\qquad w_j=0\quad(1\le j\le\beta-1).
\]
Because \(\beta\ge1\), the first and last grid indices are distinct, and \(\beta\ne0\). % lean: hne
For the zeroth moment,
\[
\sum_{j=0}^{\beta}w_j p_j^0
=
(-1)+1
=
0.
\] % lean: hw
Now fix \(1\le\ell\le\beta\). The equal-spacing schedule with \(q=1\) has \(p_0=0\) and \(p_\beta=1\). Hence
\[
\sum_{j=0}^{\beta}w_j p_j^\ell
=
-0^\ell+1^\ell
=
1,
\]
with all interior terms vanishing because their weights are zero. Thus \(w\in W_\beta(p^{\mathrm{eq}}(\beta,1))\). % lean: no_extrapolation_boundary

It remains to bound the amplification. By \cref{obj:def:amplification-criterion}, using this feasible \(w\) gives
\[
A_\beta\!\left(p^{\mathrm{eq}}(\beta,1)\right)
\le
\left(\sum_{j=0}^{\beta}|w_j|\right)^2.
\] % lean: csInf_le
The absolute values contribute only at \(j=0\) and \(j=\beta\), so
\[
\sum_{j=0}^{\beta}|w_j|=1+1=2.
\] % lean: hsum_abs
Therefore
\[
A_\beta\!\left(p^{\mathrm{eq}}(\beta,1)\right)\le 2^2=4.
\] % lean: no_extrapolation_boundary
\end{proof}
