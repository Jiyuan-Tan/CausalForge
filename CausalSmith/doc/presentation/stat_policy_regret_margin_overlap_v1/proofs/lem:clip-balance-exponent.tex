\begin{proof}[Proof of \cref{obj:lem:clip-balance-exponent}]
First suppose \(\gamma>0\), and write \(s=s_{\mathrm{feas}}\), \(t=t_{\mathrm{feas}}\), \(q_n=q_0n^{-s}\), and \(u_n=\bar u n^{-t}\).  By \cref{obj:def:feasible-rate},
\[
r_{\mathrm{feas}}=\min\{r_\star(\alpha,\gamma),g_{\mathrm{joint}}\}.
\]
Thus \(r_{\mathrm{feas}}\le r_\star(\alpha,\gamma)\) and \(r_{\mathrm{feas}}\le g_{\mathrm{joint}}\).  The selected pair \((s,t)\) realizes the value \(g_{\mathrm{joint}}=\phi(s,t)\), so \(g_{\mathrm{joint}}\) is no larger than each entry in the minimum defining \(\phi(s,t)\):
\[
g_{\mathrm{joint}}\le A_\alpha(1-2s),\quad
g_{\mathrm{joint}}\le c-s,\quad
g_{\mathrm{joint}}\le a+\frac{s}{2\gamma}+\frac{\alpha t}{2},\quad
g_{\mathrm{joint}}\le 2a-t.
\]
% lean: feasibleMaximizer_value
For \(n\ge1\), the elementary monotonicity rule \(r\le e\Rightarrow n^{-e}\le n^{-r}\) converts these exponent inequalities into termwise bounds. % lean: nat_rpow_neg_le_of_le
Work on the eventual set on which the polynomial nuisance bounds hold and, separately, \(r_{\mu,n}\ge0\).  Then
\[
n^{-r_\star(\alpha,\gamma)}\le n^{-r_{\mathrm{feas}}},
\]
\[
(nq_n^2)^{-A_\alpha}
=(q_0^2)^{-A_\alpha}n^{-A_\alpha(1-2s)}
\le (q_0^2)^{-A_\alpha}n^{-r_{\mathrm{feas}}},
\]
\[
\frac{r_{\mu,n}r_{e,n}}{q_n}
\le \frac{C_{\mathrm{prod}}}{q_0}n^{-(c-s)}
\le \frac{C_{\mathrm{prod}}}{q_0}n^{-r_{\mathrm{feas}}},
\]
\[
r_{\mu,n}u_n^{\alpha/2}q_n^{1/(2\gamma)}
\le C_\mu \bar u^{\alpha/2}q_0^{1/(2\gamma)}
n^{-\{a+s/(2\gamma)+\alpha t/2\}}
\le C_\mu \bar u^{\alpha/2}q_0^{1/(2\gamma)}
n^{-r_{\mathrm{feas}}}.
\]
For the squared nuisance term, the squaring step also uses the eventual nonnegativity \(r_{\mu,n}\ge0\) together with \(C_\mu\ge0\), so that \(r_{\mu,n}\le C_\mu n^{-a}\) implies \(r_{\mu,n}^2\le C_\mu^2 n^{-2a}\).  Hence
\[
\frac{r_{\mu,n}^2}{u_n}
\le \frac{C_\mu^2}{\bar u}n^{-(2a-t)}
\le \frac{C_\mu^2}{\bar u}n^{-r_{\mathrm{feas}}}.
\]
% lean: clip_balance_exponent
Summing the five displayed inequalities gives, for a finite constant \(C\),
\[
n^{-r_\star(\alpha,\gamma)}+(nq_n^2)^{-A_\alpha}
+\frac{r_{\mu,n}r_{e,n}}{q_n}
+r_{\mu,n}u_n^{\alpha/2}q_n^{1/(2\gamma)}
+\frac{r_{\mu,n}^2}{u_n}
\le C n^{-r_{\mathrm{feas}}}(\log n)^0.
\]
Since \(r_{\mathrm{feas}}=\min\{r_\star(\alpha,\gamma),g_{\mathrm{joint}}\}\), this is the desired positive-\(\gamma\) exponent bound with logarithmic power \(p=0\).

Now suppose \(\gamma=0\).  \Cref{obj:def:feasible-rate} gives
\[
r_{\mathrm{feas}}=\min\{A_\alpha,c\}.
\]
Again \(r_{\mathrm{feas}}\le A_\alpha\) and \(r_{\mathrm{feas}}\le c\).  On the eventual set where the polynomial product bound holds, for \(n\ge1\),
\[
n^{-A_\alpha}\le n^{-r_{\mathrm{feas}}},
\qquad
r_{\mu,n}r_{e,n}\le C_{\mathrm{prod}}n^{-c}\le C_{\mathrm{prod}}n^{-r_{\mathrm{feas}}}.
\]
Thus the strict-overlap terms satisfy, for a finite constant \(C\),
\[
n^{-A_\alpha}+r_{\mu,n}r_{e,n}
\le C n^{-r_{\mathrm{feas}}}(\log n)^0
= C n^{-\min\{A_\alpha,c\}}(\log n)^0.
\]
% lean: clip_balance_exponent
\end{proof}
