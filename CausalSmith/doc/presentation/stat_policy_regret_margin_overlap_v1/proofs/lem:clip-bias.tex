\begin{proof}[Proof of \cref{obj:lem:clip-bias}]
Fix a bounded measurable covariate test function \(\phi\), and write \(\bar e_q(x)=e_q(x;\bar\eta)\).  Since \(q\in(0,1)\), the clipped denominators \(\bar e_q\) and \(1-\bar e_q\) are bounded away from zero; the bounded-outcome and bounded-nuisance assumptions make all terms below integrable.  Expanding \cref{obj:def:clipped-aipw-score} gives
\[
\begin{aligned}
\int \phi(X)\Gamma_q(O;\bar\eta)\,dP
&=\int \phi(\bar\mu_1-\bar\mu_0)\,dP_X  \\
&\quad+\int e_P\mu_1\,\frac{\phi}{\bar e_q}\,dP_X
      -\int e_P\bar\mu_1\,\frac{\phi}{\bar e_q}\,dP_X \\
&\quad-\int (1-e_P)\mu_0\,\frac{\phi}{1-\bar e_q}\,dP_X
      +\int (1-e_P)\bar\mu_0\,\frac{\phi}{1-\bar e_q}\,dP_X .
\end{aligned}
\]
Here the treated and control terms have been converted to \(P_X\)-integrals using the defining conditional-mean relations for \(e_P,\mu_1,\mu_0\). % lean: clippedAIPWScore_test_expand_pointwise % lean: hscore_split % lean: hI0 % lean: hI1 % lean: hI2 % lean: hI3 % lean: hI4

Collecting the integrand pointwise and using \(\tau_P=\mu_1-\mu_0\), \(\Delta_a=\bar\mu_a-\mu_a\), one obtains
\[
\int \phi(X)\Gamma_q(O;\bar\eta)\,dP
-
\int \phi(x)\tau_P(x)\,dP_X(x)
=
\int \phi(x)\,
(\bar e_q(x)-e_P(x))
\left\{
\frac{\Delta_1(x)}{\bar e_q(x)}
+
\frac{\Delta_0(x)}{1-\bar e_q(x)}
\right\}\,dP_X(x).
\]
Since this holds for every bounded measurable \(\phi\), it is precisely the displayed conditional-mean drift identity. % lean: clipBias_integrand_collect_pointwise % lean: hpx_collect % lean: clip_bias

The same formula gives the cancellation mechanisms recorded in the statement: if \(\bar e_q(x)=e_P(x)\), the leading factor is zero; if \(\Delta_0(x)=\Delta_1(x)=0\), the bracket is zero. % lean: clip_bias

Finally, the assumptions alone do not allow one to replace this algebra by a cancellation claim on the region \(p_P(x)>q\).  There is a well-formed bounded positive law on \(\mathbb R\), with \(q'=1/4\), plug-ins \(m_0\equiv0\), \(m_1\equiv1\), \(e'\equiv0\), and \(x=0\), for which \(q'<p_{P'}(0)\) but the displayed drift is nonzero.  Thus no additive clipped-region bound of the asserted form follows from the frozen assumptions alone. % lean: clipBiasCounterLaw_overlap % lean: clipBiasCounterLaw_bias_ne % lean: clip_bias
\end{proof}
