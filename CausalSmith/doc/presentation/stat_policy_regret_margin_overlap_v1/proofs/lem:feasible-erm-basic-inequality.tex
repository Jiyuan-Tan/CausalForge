\begin{proof}[Proof of \cref{obj:lem:feasible-erm-basic-inequality}]
The proof works with the countable dense skeleton \(\Pi_0=\{\pi_j\}\) from \cref{obj:ass:policy-class}.  For a realized sample, write \(s_j=\widehat V_{n,q}(\pi_j)\).  The set of \(1/n\)-near maximizers among the indices is nonempty: the finite-sample scores are bounded above by \(n^{-1}\sum_i|\Gamma_i|\), so an element within \(1/n\) of the supremum exists. % lean: feasibleERM_nearSet_nonempty
The estimator chooses the least such index.  Lean proves measurability by characterizing the fiber \(\{j_n=j\}\) as the event that \(j\) is near-maximizing and no smaller index is near-maximizing; these are countable intersections of measurable score comparisons. % lean: feasible_erm_basic_inequality
Since each enumerated policy belongs to \(\Pi\), the selected policy is \(\Pi\)-valued. % lean: feasible_erm_basic_inequality

It remains to pass from near-maximality over \(\Pi_0\) to every comparator in \(\Pi\).  Given \(\pi^b\in\Pi\), density supplies a sequence \(\pi_{j_t}\) converging pointwise to \(\pi^b\).  On the finite observed covariate set \(\{X_i\}_{i=1}^n\), this convergence is eventually exact at every sample point, hence \(\widehat V_{n,q}(\pi_{j_t})=\widehat V_{n,q}(\pi^b)\) for some \(t\). % lean: feasible_erm_basic_inequality
Applying the selected index's \(1/n\)-near-maximality to \(j_t\) gives
\[
\widehat V_{n,q}(\pi^b)
\le
\widehat V_{n,q}(\widehat\pi_n)+\frac1n .
\]
Equivalently,
\[
P_n\!\left[(\widehat\pi_n-\pi^b)\Gamma_q\right]\ge -\frac1n .
\]
% lean: feasible_erm_basic_inequality
Under \cref{obj:ass:optimal-in-class}, \(\pi^\star_P\in\Pi\), so the same comparator inequality applies with \(\pi^b=\pi^\star_P\). % lean: feasible_erm_basic_inequality
\end{proof}
