\begin{proof}[Proof of \cref{obj:lem:rho-deficient-regime-algebra}]
Because \(s>0\) and \(4s<d\),
\[
1<\frac d{4s}.
\]
Thus
\[
2+\frac1\alpha
<
1+\frac d{4s}+\frac1\alpha,
\]
with positive denominators, and so
\[
\frac{2}{1+d/(4s)+1/\alpha}
<
\frac{2}{2+1/\alpha}.
\]
Using
\[
\frac{2}{2+1/\alpha}
=
\frac{2\alpha}{2\alpha+1},
\]
we obtain the strict exponent comparison
\[
\frac{2}{1+d/(4s)+1/\alpha}
<
\frac{2\alpha}{2\alpha+1}.
\]
% lean: rho_deficient_regime_algebra
It remains to identify the maximum in \cref{obj:def:published-hoif-rate}. If \(n=1\), both rate terms are \(1\). If \(n>1\), the strict exponent inequality above implies, after negating exponents and using monotonicity of \(x\mapsto n^x\),
\[
n^{-2\alpha/(2\alpha+1)}
\le
n^{-2/(1+d/(4s)+1/\alpha)}.
\]
Hence the maximum is the deficient-covariate term, namely
\[
\rho_n
=
n^{-2/(1+d/(4s)+1/\alpha)}.
\]
Together with the strict comparison already proved, this is the desired conclusion. % lean: rho_deficient_regime_algebra
\end{proof}
