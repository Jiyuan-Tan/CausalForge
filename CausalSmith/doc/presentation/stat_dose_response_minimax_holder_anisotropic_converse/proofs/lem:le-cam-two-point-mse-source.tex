\begin{proof}[Proof of \cref{obj:lem:le-cam-two-point-mse-source}]
Let \(K_+=\max\{K,0\}\) and set
\[
c_K=\frac{\exp(-K_+)}{32}>0 .
\]
Fix any measurable estimator \(T\) for which the two squared losses below are integrable, and put
\(r=|\theta_1-\theta_0|/2\). Since \(2r\le |\theta_0-\theta_1|\), the two-point testing reduction gives
\[
\frac{1-\mathrm{TV}(Q_0,Q_1)}{2}
\le
\max\Bigl\{
Q_0(|T-\theta_0|\ge r),
Q_1(|T-\theta_1|\ge r)
\Bigr\}.
\]
% lean: Causalean.Stat.half_one_sub_tvDist_le_max_error
The KL-budget assumption is an inequality against the finite extended-real number corresponding to \(K\). Hence the divergence is finite, the absolute-continuity side condition for Bretagnolle--Huber follows, and the real value of the divergence is at most \(K_+\). Bretagnolle--Huber gives
\[
\frac12\exp\{-\mathrm{KL}(Q_0,Q_1)\}
\le 1-\mathrm{TV}(Q_0,Q_1).
\]
Together with \(\mathrm{KL}(Q_0,Q_1)\le K_+\), this yields
\[
\frac{\exp(-K_+)}{4}
\le
\max\Bigl\{
Q_0(|T-\theta_0|\ge r),
Q_1(|T-\theta_1|\ge r)
\Bigr\}.
\]
% lean: Causalean.Stat.bretagnolle_huber_affinity
For \(i=0,1\), on the event \(|T-\theta_i|\ge r\) one has
\(r^2\le (T-\theta_i)^2\). Since the squared loss is nonnegative,
\[
r^2 Q_i(|T-\theta_i|\ge r)
\le
E_{Q_i}\{(T-\theta_i)^2\}.
\]
Write
\[
p_i=Q_i(|T-\theta_i|\ge r),
\qquad
m_i=E_{Q_i}\{(T-\theta_i)^2\}.
\]
To pass from the two separate event bounds to the maximum, distinguish which error probability is larger. If \(p_0\le p_1\), then
\(r^2\max\{p_0,p_1\}=r^2p_1\le m_1\le\max\{m_0,m_1\}\). If \(p_1\le p_0\), the same argument with the indices reversed gives
\(r^2\max\{p_0,p_1\}\le\max\{m_0,m_1\}\). Therefore
\[
r^2\max\Bigl\{
Q_0(|T-\theta_0|\ge r),
Q_1(|T-\theta_1|\ge r)
\Bigr\}
\le
\max_{i=0,1}E_{Q_i}\{(T-\theta_i)^2\}.
\]
Combining this maximum bound with the preceding testing lower bound gives
\[
\max_{i=0,1}E_{Q_i}\{(T-\theta_i)^2\}
\ge
r^2\,\frac{\exp(-K_+)}{4}.
\]
Finally, \(r^2=(\theta_1-\theta_0)^2/4=\Delta^2/4\), so the last display implies the weaker bound
\[
\max_{i=0,1}E_{Q_i}\{(T-\theta_i)^2\}
\ge
\frac{\exp(-K_+)}{32}\,\Delta^2
=
c_K\Delta^2 .
\]
% lean: Causalean.Stat.Minimax.le_cam_two_point_mse
\end{proof}
