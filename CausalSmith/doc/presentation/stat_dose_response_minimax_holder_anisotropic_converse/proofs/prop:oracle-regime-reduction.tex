\begin{proof}[Proof of \cref{obj:prop:oracle-regime-reduction}]
\begin{enumerate}
\item By the certified sharp pointwise lower bound \cref{obj:thm:sharp-pointwise-lower-bound}, applied with the stated positivity, regime-constant, and baseline-slack hypotheses, there is a constant \(c>0\) such that, for all sufficiently large \(n\),
\[
c\, n^{-2\alpha/(2\alpha+1)}
\le
R_n\!\left(\mathcal P_{\alpha,\beta,s}(M,c_0,\varepsilon_0,t_0),t_0\right).
\]
Fix this \(c\). % lean: sharp_pointwise_lower_bound

\item We now restrict to sample sizes \(n\) in the eventual set from the preceding display and, at the same time, to the eventual set \(n\ge 1\). Thus for every sufficiently large \(n\), both the lower bound above and the elementary sample-size condition \(n\ge 1\) are available. % lean: Filter.eventually_ge_atTop

\item For such an \(n\), the smooth-covariate assumption \(d\le 4s\), together with \(\alpha>0\), \(s>0\), and \(n\ge 1\), lets us apply the oracle-regime algebra \cref{obj:lem:rho-oracle-regime-algebra}. Hence
\[
\rho_n
=
n^{-2\alpha/(2\alpha+1)} .
\]
% lean: rho_oracle_regime_algebra

\item Substituting this identity for \(\rho_n\) in the lower bound from Step 1 gives
\[
c\,\rho_n
\le
R_n\!\left(\mathcal P_{\alpha,\beta,s}(M,c_0,\varepsilon_0,t_0),t_0\right)
\]
for all sufficiently large \(n\). Together with \(c>0\) and the displayed identity for \(\rho_n\), this proves the claim. % lean: oracle_regime_reduction
\end{enumerate}
\end{proof}
