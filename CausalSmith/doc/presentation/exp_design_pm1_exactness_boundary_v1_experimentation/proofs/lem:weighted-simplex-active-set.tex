\begin{proof}[Proof of \cref{obj:lem:weighted-simplex-active-set}]
First assume \(\kappa>0\).  The admissible active-set data exist by applying the intermediate value argument to the increasing threshold function
\[
\lambda\longmapsto \sum_{i\in\mathcal I}\frac{\max\{\lambda-\alpha_i,0\}^2}{\beta_i};
\]
one obtains a nonempty support \(S=\{i:\alpha_i<\lambda\}\) and a multiplier \(\lambda\) satisfying exactly the displayed admissibility equations and inequalities. % lean: exists_admissible
For this pair, the denominator \(\sum_{h\in S}(\lambda-\alpha_h)/\beta_h\) is positive, so the displayed formula defines a nonnegative vector whose coordinates sum to \(M\); hence \(t^\star\in\Delta_M\). % lean: activeSetPoint_mem
The KKT certificate for this point gives strict optimality on the simplex.  Namely, on \(S\) the stationarity relation identifies \(\lambda-\alpha_i\) with the weighted-norm gradient, while outside \(S\) the inequalities \(\lambda\le\alpha_j\) and \(s_j\ge0\) give the complementary nonnegative remainder.  Strict Cauchy--Schwarz for any \(s\ne t^\star\) then yields \(\Phi(t^\star)<\Phi(s)\). % lean: activeSetPoint_strict_min
The same stationarity identity, multiplied by \(t_i^\star\) and summed over \(i\), gives
\[
\Phi(t^\star)=\lambda\sum_i t_i^\star=M\lambda .
\] % lean: activeSetPoint_value

It remains in the positive-\(\kappa\) case to see uniqueness of \((S,\lambda)\).  If another admissible pair \((S',\lambda')\) had the stated properties, its active-set point would also be a strict minimizer, so the two points must coincide; otherwise each would have strictly smaller value than the other. % lean: activeSetPoint_strict_min
The support is recovered from the point by the equivalence \(i\in S\) if and only if \(t_i^\star>0\), so \(S=S'\). % lean: activeSetPoint_pos_iff
Finally the value identity gives \(M\lambda=M\lambda'\), and \(M>0\) implies \(\lambda=\lambda'\). % lean: activeSetPoint_value

Now assume \(\kappa=0\).  Then \(\Phi(t)=\sum_i\alpha_i t_i\) is linear.  Choose a coordinate \(k\) with minimal \(\alpha_k\).  If \(t\) is a minimizer, comparison with the vertex \(M e_k\) gives
\[
\sum_i(\alpha_i-\alpha_k)t_i\le0,
\]
while each summand is nonnegative; hence every nonzero coordinate of \(t\) lies among the \(\alpha\)-minimizing coordinates.  Conversely, any simplex point supported only on those minimizing coordinates has value \(M\alpha_k\), and every simplex point has value at least \(M\alpha_k\).  This is exactly the exposed \(\alpha\)-minimizing face. % lean: kappa_zero_face
\end{proof}
