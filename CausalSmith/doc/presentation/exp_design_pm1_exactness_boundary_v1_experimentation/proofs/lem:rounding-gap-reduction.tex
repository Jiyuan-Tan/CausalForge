\begin{proof}[Proof of \cref{obj:lem:rounding-gap-reduction}]
Set \(q=2(m-1)\).  Under \cref{obj:ass:two-block-homophily} we have \(m\ge2\).  The change of variables is invertible on the reduced triangle: given \((x,y,z_{\mathrm{sp}})\in T_m\), take
\[
u=1-x,\qquad v=\frac{z_{\mathrm{sp}}-y}{2m}.
\]
Using \(qx+y+z_{\mathrm{sp}}=2m\), these formulas recover
\[
1-u=x,\qquad
1+(m-1)u-mv=y,\qquad
1+(m-1)u+mv=z_{\mathrm{sp}} .
\] % lean: coord_inv

The image of the block elliptope under \(F_{r,\kappa}\) is therefore exactly the set of reduced objective values over \(T_m\).  In one direction, \cref{obj:lem:block-spectral-coordinates} sends any \(X(u,v)\in\mathcal E_m^{\mathrm{blk}}\) to a point of \(T_m\) and identifies \(F_{r,\kappa}(X(u,v))\) with \(\phi_{r,\kappa}(x,y,z_{\mathrm{sp}})\).  In the other direction, the inverse coordinates above produce a block matrix in \(\mathcal E_m^{\mathrm{blk}}\) with the same reduced value. % lean: hRel

Likewise, the image of \(\mathcal C_m^{\pm}\) under \(F_{r,\kappa}\) is exactly the set of reduced objective values over the parity-truncated slice \(T_m\cap\{y+z_{\mathrm{sp}}\ge d_m\}\).  If \(X\in\mathcal C_m^{\pm}\), block exchangeability gives parameters \(u,v\) for its second moment, and \cref{obj:lem:pm-reduced-slice-characterization} gives both the reduced-triangle constraints and the parity inequality; \cref{obj:lem:block-spectral-coordinates} again identifies the value with \(\phi_{r,\kappa}\). % lean: secondMoment_blockSym_of_exchangeable
Conversely, any \((x,y,z_{\mathrm{sp}})\in T_m\) satisfying \(d_m\le y+z_{\mathrm{sp}}\) is converted by the same inverse coordinates into a block matrix \(X(u,v)\), and \cref{obj:lem:pm-reduced-slice-characterization} places it in \(\mathcal C_m^{\pm}\) with the corresponding reduced objective value. % lean: pm_reduced_slice_characterization

Substituting these two image descriptions into \cref{obj:def:implementability-gap} gives
\[
\Delta_m^{\pm}(r,\kappa)
=
\inf_{\substack{(x,y,z_{\mathrm{sp}})\in T_m\\ y+z_{\mathrm{sp}}\ge d_m}}
\phi_{r,\kappa}(x,y,z_{\mathrm{sp}})
-
\inf_{(x,y,z_{\mathrm{sp}})\in T_m}
\phi_{r,\kappa}(x,y,z_{\mathrm{sp}}),
\]
as claimed. % lean: rounding_gap_reduction
\end{proof}
