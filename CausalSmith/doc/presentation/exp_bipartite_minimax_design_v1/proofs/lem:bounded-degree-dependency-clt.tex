\begin{proof}[Proof of \cref{obj:lem:bounded-degree-dependency-clt}]
Discard a finite initial segment so that \(v_n\geq cn>0\).  Define
\[
Y_{n,i}=\frac{X_{n,i}}{\sqrt{v_n}}.
\]
Scaling by a deterministic constant preserves the dependency graph, the summands remain centered, and their sum has second moment one:
\[
\mathbb E\!\left[\left(\sum_{i\in\iota_n}Y_{n,i}\right)^2\right]=1.
\]
% lean: Causalean.SteinMethod.depGraph_div_const

Moreover,
\[
|Y_{n,i}|
 \leq \frac{M}{\sqrt{v_n}}
 \leq \frac{M/\sqrt c}{\sqrt n}
 =:b_n.
\]
Thus \(b_n\to0\) and
\[
n b_n^3=\frac{(M/\sqrt c)^3}{\sqrt n}\longrightarrow0.
\]
% lean: Causalean.SteinMethod.bounded_degree_dependency_clt_of_variance_floor_all
The dependency degree is still bounded by \(D\), so the dependency-graph Stein central limit theorem applies to the standardized array.  It gives, for every \(s\in\mathbb R\),
\[
\Pr\!\left(\frac{S_n}{\sqrt{v_n}}\leq s\right)
\longrightarrow
\Pr(Z\leq s),
\qquad Z\sim\mathcal N(0,1).
\]
% lean: Causalean.SteinMethod.stein_cdf_clt_of_depGraph
% lean: Causalean.SteinMethod.bounded_degree_dependency_clt
% lean: CausalSmith.Experimentation.BipartiteMinimaxDesign.bounded_degree_dependency_clt
\end{proof}
